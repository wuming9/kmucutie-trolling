# Offshore Spreader Bar System  
### Trolling Distribution Frame Assembly – OEM Production

The Spreader Bar System is a structured offshore trolling assembly designed to create a wide bait-school profile and stable surface disturbance.  
Compared to linear rigs, spreader bars require tighter assembly tolerance and stronger material control to ensure consistent drag, tracking, and durability.

KMUCUTIE provides OEM production for complete spreader bar assemblies, including bar structure, teaser layout, leader system, terminal rigging, and private label packaging.

---

## 1. System Structure Overview

A standard spreader bar system includes:

- Spreader bar frame (fixed or folding)
- Center hub and connection points
- Dropper leaders (left / right layout)
- Teaser bodies (squid / birds / soft bodies / flashers)
- Mainline leader, swivel, and terminal connection
- Optional hooked center lure or hookless teaser configuration

All rigs are assembled and inspected before shipment.

---

## 2. Bar Structure Options

OEM structural options:

- **Bar length**: multiple size ranges available (small / medium / large)
- **Frame type**: fixed bar / folding bar (transport-friendly)
- **Material**: corrosion-resistant components suitable for saltwater use
- **Connection design**: reinforced eyes, balanced spacing, stable tracking

Folding structure is recommended for distributors and export cartons to reduce shipping volume.

---

## 3. Teaser Layout & Configuration

Customizable layout items:

- Number of teaser positions (common: 4–12; custom available)
- Left/right spacing and symmetry control
- Teaser body type mix:
  - Soft squid
  - Octopus bodies
  - Bird teasers
  - Feather teasers
  - Flash materials (by request)

Color catalog can be provided for selection and repeatable production.

---

## 4. Leader & Rigging Specification

Rigging variables OEM customers may define:

- Main leader length
- Dropper leader length distribution (outer vs inner)
- Leader diameter / strength requirement
- Crimp type and compression standard
- Swivel grade and corrosion resistance
- Optional wire leader / reinforcement for high-bite scenarios

Assembly consistency is critical for tracking and drag stability.

---

## 5. Production & Assembly Capability

KMUCUTIE supports batch production for:

- Bar frame preparation and structural checks
- Dropper leader cutting and length standardization
- Crimped assembly and pull-strength inspection
- Teaser body installation and layout verification
- Final inspection and packaging

Stable production workflow supports repeat orders with consistent structure and layout.

---

## 6. Quality Control Points

Key QC checks include:

- Frame straightness / folding joint stability (if folding)
- Symmetry of left-right layout
- Dropper length tolerance control
- Crimp integrity and alignment
- Swivel rotation and terminal strength
- Final assembly cleanliness and label correctness

QC is performed before packing to reduce distributor rework.
