# Quality Control Overview

We prioritize repeatability and consistency for repeat orders.

---

## Incoming Inspection

- Body material verification
- Hook brand verification
- Split ring specification confirmation

---

## In-Process Control

- Paint finish consistency
- Weight consistency
- Hook installation inspection
- Split ring strength check

---

## Performance Testing

- Swimming action test (random sampling)
- Balance verification
- Corrosion resistance testing (if required)

---

## Pre-Shipment Inspection

- Packaging check
- Label verification
- Carton reinforcement inspection

Third-party inspection is welcome if required by client.
